Builder- this is where you define the Recepie.

select Builder from top menu

1. Type recepie title

2. Select from the equipmnt list (click/ unclick) the asset required for this recepie

3. change the qty for each of the selected Assets

steps:

Type Step description

Add instructions Load images for this step.

click next step / prev step to move between steps. once done click on the finish button

Viewer - this is where you use the recepie.

select Viewer from top menu

Select Recepie from the the list (click to select).

you can retun to the Recepie list by clicking on the button on the top right side
